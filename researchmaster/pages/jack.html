Together with some traditional inference algorithms the reformulation algorithms accessible from the below portals are the beginning of the system Jack. Jack is an approach to logical inference that analyzes a given inference problem, chooses the automated reasoning tool that is most appropriate for solving that problem, transforms the problem into the language that tool requires, and finally invokes that tool to solve the inference problem.

The premise is that each automated reasoning tool was developed to address a unique class of problems, and if a system could automatically detect that class and reformulate the problem as appropriate, it would exhibit better performance properties than any one of the individual systems. So far, there is just one nontrivial reformulation algorithm, what-to-how, that transforms classical logic (Finite Herbrand Logic) queries into Logic Programming (nonrecursive datalog with negation) queries.

  • Reformulate: reformulate sentences using a variety of tools.
  • Classical: ask various queries about classical logic.
  • Logic Programming: ask various queries about logic programming languages.

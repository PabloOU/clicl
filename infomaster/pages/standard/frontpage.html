Infomaster

  

Welcome to Infomaster. Infomaster is a Deductive Dataweb Management System operated as a web-based service. It provides support for users to access and update structured data as well as logical rules relevant to that data.

While the system can be used in stand-alone fashion, it can also coordinate with other dataweb services to provide integrated access to information from multiple sites. When used in this way, Infomaster is able to give its users the illusion of a centralized, homogeneous information system. Information consumers can ask questions, confident that the system will provide information from all relevant information sources. Authorized information suppliers can make updates, confident that the system will distribute those updates appropriately.

Infomaster's commitment to distributed architecture begins within the system itself. Infomaster is implemented as a collection of communicating "agents", each providing different capabilities and each communicating with its peers via message-passing. When Infomaster is first started, it contains several predefined agents, all running within a single process on the host machine. The system administrator can create additional agents and can kill existing agents. In addition, agents running on other computer systems can be linked into Infomaster, thereby creating a distributed dataweb system spanning the Internet.

In the current version of Infomaster, there are seven predefined local agents when an instance of the system is created. The functions of these agents are summarized in the following table. Click on the links in the table to view metadata about these agents. Click on the links on the button bar above to access the agents themselves.

Agent Class Description
Standard Interface This agent is a user-level interface to the Integrator.
Integrator Facilitator This agent integrates data from multiple sources.
Library Ruleserver This agent contains rules used by the Integrator.
Warehouse Dataserver This agent contains data used by the Integrator.
Repository Fullserver This agent contains both data and rules in one place.
Manager Fastserver This agent contains metadata used by system.
Metalibrary Ruleserver This agent contains metarules used by system.

These agents are capable of performing a large selection of operations appropriate to information management. Some of these operations can be invoked via links in the headers associated with each agent. Some operations can be invoked by clicking on icons appearing on various web pages. All of the operations can be invoked from the Operations page.
